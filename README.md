## 🛠️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/pratapgiri/react.git
```

2. Navigate to the project folder:

```
cd Reactjs
```

3. Install dependencies:

```
npm install
```

4. Start the development server:

```
npm run dev
```
